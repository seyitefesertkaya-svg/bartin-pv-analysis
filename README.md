# Photovoltaic Power Plant Feasibility Analysis (Port Project)

This project presents a technical and economic feasibility analysis of a photovoltaic (PV) power plant designed for port self-consumption.

#  Overview
The system is designed to meet the electricity demand of a port facility using solar energy.

# Analysis Includes
- System sizing and energy production estimation
- Economic analysis (NPV, IRR, LCOE)
- Cost modeling (CAPEX, OPEX)
- Efficiency and performance evaluation

# Data Sources
- NASA POWER (GHI data)
- Real-world energy consumption estimations

# Tools Used
- Python
- MATLAB
- LaTeX

# Results
- Financial feasibility evaluated using standard energy metrics
- System optimized for maximum energy efficiency
